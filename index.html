Расчет железобетонных конструкций

Укажите размеры

Логи: 
Актуальность настоящей работы обусловлена необходимостью создания доступного облачного сервиса
             для проведения предварительного расчета и оценки при проектировании железобетонной конструкции.
Объектом исследования является проектирование железобетонных конструкций.
Предмет исследования – методики расчета конструкции и доступные средства
                    для выполнения расчета железобетонных конструкций.
Цель – теоретический анализ методик расчета железобетонных конструкций
                    и разработка облачного сервиса производящего расчет.
Научная новизна работы состоит в реализации технических возможностей
                    платформы ASP.Net и среды разработки Visual Studio при выполнении расчета железобетонных конструкций.
Практическая значимость работы заключается в создании кросс-браузерного
                    онлайн-ресурса предназначенного для выполнения расчетов железобетонных конструкций.
Задачи, которые необходимо решить в ходе выполнения работы:
- описать существующие методики и расчета конструкций;
- определить вероятную методику расчета;
- проанализировать  и определить средства разработки;
- на основе анализа средств выявить язык программирования;
- выявить требования к системе, предоставляющей средства расчета;
- разработать структуру сайта;
- изучить среду разработки;
- на основе данной среды создать прототип сайта;
- протестировать данную систему.
